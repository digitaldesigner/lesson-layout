    Welcome to Lesson Layout (GitHub) — a simple web layout for displaying lists, lessons, to-do's, release notes and anything that works in a horizontal format.

    • üöÄ Easy — It's lightweight and easy to build on, no packages or dependencies. Just plain ol' HTML and CSS with a little JavaScript.

    üóÇÔ∏è Cards

    Regardless of how much content is contained, each card can be expanded with a click or tap. To close, press ESC or click the cross. There is a layout calculation to make sure there's always a little part of the next card visible, so horizontal scroll is obvious to the user — and scroll snapping to make sure cards are always perfectly centered.

    ‚úÖ Mark as Done

    Each card can be marked as complete, which will save its status to localStorage. If cards are completed, the page will be loaded to the next incomplete card, to save the user from scrolling endlessly!

    Theming

    The light and dark themes are semantically split and easy to theme using the separate --var styles. Out of the box the theme is accessible, mobile-friendly and cross-browser compliant.

    Written in LESS, you can make any style adjustments in less/_variables.less
